### Exercício
Você vai receber um objeto `window` e um objeto `document`, nesta ordem.
Ver o arquivo `index.html`, sua função deverá retornar o conteúdo da tag `p`.

### Entrada exemplo
``` html
<!DOCTYPE html>
<html lang="en">
<head>
    <title>Document</title>
</head>
<body>
    <p id="paragraph">HEHEHE!</p>
</body>
</html>
```

### Retorno esperado
```
HEHEHE!
```
